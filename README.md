# How to run the app

1. Install node
2. Install sass
3. Run command:
'npm install'
4. Change permissions on launch file if necessary with command: 
'chmod 755 ./scripts/launch_server.sh'
5. Run command:
'npm start'
6. Open http://localhost:3000
